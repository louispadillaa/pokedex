La aplicación muestra el listado y el detalle de los Pokémon de las primeras 3 generaciones.

La imagen que representa un Pokémon en el listado muestra las variaciones que estos tuvieron durante las primeras versiones, desde la versión Green (1996) hasta la version Emerald (2005).

Los detalles de un Pokémon individual muestra sus estadísticas base y los registros de la Pokédex de las diferentes versiones.

El proyecto fue desarrollado usando la librería de JavaScript [Angular](https://angular.io/) para crear la interfaz de usuario, en comunicación con la Api RESTful [PokéAPI](https://pokeapi.co/).

## Requisitos mínimos

- [Nodejs](https://nodejs.org) con soporte de largo plazo (LTS).
- Un navegador web

## Ambiente de pruebas

Ejecutar en la raíz del proyecto:

```
npm start
```

# Creación de Cuenta de Azure

### Requisitos previos

- Correo institucional universitario activo (Tecnológico Comfenalco)
- Navegador web actualizado
- Conexión a internet

### Paso 1 — Acceder al portal de Azure for Students

1. Abrí el navegador y fui a [https://azure.microsoft.com/es/free/students](https://azure.microsoft.com/es/free/students)
2. Hice clic en el botón **"Empiece gratis"**
3. Se abrió la pantalla de inicio de sesión de Microsoft y coloqué mi usuario 

### Paso 2 — Iniciar sesión con correo institucional

1. Ingresé con mi correo institucional universitario
2. Hice click en **"Siguiente"**
3. Ingresé mi contraseña institucional
4. Completa la verificación en dos pasos si aplica

### Paso 3 — Verificación de elegibilidad estudiantil

1. Microsoft verifica que el dominio pertenece a una institución educativa acreditada así que cumplí con esa sección colocando la universidad a la cual pertenezco y otros campos asociados a la univerdad.
2. Solicité aceptar los términos y condiciones de **Azure for Students**
3. Leí y acepté los términos haciendo clic en **"Aceptar"**

### Paso 4 — Activación de la suscripción

1. Azure activa automáticamente la suscripción **Azure for Students**
2. Pude ingresar al portal: [https://portal.azure.com](https://portal.azure.com)
3. En suscripciones vi como tenía los **$100 USD** de crédito disponibles

| Crédito disponible | $100 USD |
| Vigencia | 12 meses renovables |
| Tarjeta de crédito requerida | No |
